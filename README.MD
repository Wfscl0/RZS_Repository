# RZS_PCBs

RZS 电气系统相关 PCB 设计、仿真、测试资料与规则文档归档。

## 内容

- `BSPD`：BSPD 相关设计资料
- `EBS`：EBS 相关设计资料
- `IMD-BMS错误锁存`：IMD/BMS 错误锁存电路
- `TSAL`：TSAL 设计与仿真
- `低压盒`、`基板`、`继电器检测`、`测试用电源`：各功能板设计资料
- `spice`、`所有板子完整仿真（目前已过时）`：电路仿真及历史资料
- `规则`、`车检表`、`2025电检结果`：赛事规则、检查表及测试记录
- 其他目录：往届资料、组间协作资料和课程作业等

## 使用说明

仓库包含原理图、PCB 工程、仿真模型、PDF、压缩包及部分较大的二进制文件。克隆前请确认本地磁盘空间充足：

```bash
git clone https://github.com/Wfscl0/RZS_PCBs.git
```

部分工程需要对应的 EDA 或仿真软件打开。历史目录中的内容可能已经过时，使用前请核对当前设计和赛事规则。
